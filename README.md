## Data
#### Data source
We are training a large language model using a corpus of public domain novels from Project Gutenberg
[Gutenberg](https://www.gutenberg.org/)
#### Datasets
For the dataset, we used fiction books provided through the Hugging Face library, leveraging its built-in functions for loading and processing. 
The dataset is available at
[Gutenberg-BookCorpus-Cleaned-Data-English](https://huggingface.co/datasets/incredible45/Gutenberg-BookCorpus-Cleaned-Data-English)
